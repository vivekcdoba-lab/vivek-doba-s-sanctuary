# Glossary

| Term | Meaning |
|---|---|
| **VDTS** | Vivek Doba Training Solutions — the brand |
| **Seeker / Sadhak** | The customer (user) — both terms used interchangeably |
| **Coach / Guruji** | A trained coach; "Guruji" is the honorific used for Vivek Doba |
| **LGT** | Life's Golden Triangle — the four-pillar framework (Dharma, Artha, Kama, Moksha) |
| **Dharma** | Pillar 1 — purpose, righteousness, duty |
| **Artha** | Pillar 2 — wealth, livelihood, business |
| **Kama** | Pillar 3 — desire, relationships, family |
| **Moksha** | Pillar 4 — liberation, spirituality |
| **IGS** | Initial Goal Setting — the LGT assessment that counts as Session #1 |
| **Sampoorna Din** | "Complete day" score on the daily worksheet |
| **Sankalp** | Daily intention/resolution set the night before |
| **Vishnu Protocol** | The 6-month personalized transformation roadmap |
| **MOOCH** | Indian-context personality dimensions (Overthinking, Negativity, Comparison, Fear, Attachment, Resistance) |
| **FIRO-B** | Schutz's Fundamental Interpersonal Relations Orientation — Behavior framework |
| **PERMA-H+** | The 8-domain happiness framework used in `happiness_assessments` |
| **Purushaarthas** | Sanskrit term for the four aims of life — used as a separate alias-assessment |
| **Wheel of Life (WoL)** | 8-domain life balance check |
| **Joint payment** | A single payment shared across linked seekers (`seeker_links` group with `is_joint=true`) |
| **Seeker Link** | An admin-managed link between two seekers for billing/visibility — never shares assessments or sessions |
| **Coach Seeker** | A row in `coach_seekers` connecting a coach to a seeker; `is_primary` marks the lead coach |
| **Excused session** | Attendance state that does NOT consume a paid session slot |
| **Danger Zone** | Coach view of seekers with red traffic-light health |
| **Sacred Sound** | Procedural Web Audio API system for ambient/temple sounds + Mood Mixer |
| **Operation Docs** | This documentation hub at `/admin/operation-docs` |
