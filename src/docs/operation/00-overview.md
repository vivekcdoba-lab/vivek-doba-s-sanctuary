# Overview

**Vivek Doba Training Solutions (VDTS)** is a spiritual coaching platform integrating the **Life's Golden Triangle (LGT)** framework — Dharma, Artha, Kama, Moksha — with structured coaching, daily practice, and business mentorship.

## Identity

- **Brand**: Vivek Doba Training Solutions (VDTS)
- **Coach / Guruji**: Vivek Doba
- **Framework**: Life's Golden Triangle (LGT) — Dharma · Artha · Kama · Moksha
- **Audience**: Seekers (Sadhaks) — primarily entrepreneurs and professionals seeking integration of spirituality and business
- **Domain**: [vivekdoba.com](https://vivekdoba.com)

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite 5 + TypeScript 5 + Tailwind CSS v3 |
| Routing | react-router-dom v6 |
| State | Zustand (auth, UI), TanStack Query (server state) |
| UI Kit | shadcn/ui + lucide-react icons + Recharts |
| Backend | Lovable Cloud (Supabase under the hood) — Postgres, Auth, Storage, Edge Functions, Realtime |
| AI | Lovable AI Gateway (Gemini, GPT, image models) |
| Email | Resend |
| SMS / WhatsApp | Twilio |
| PWA | Installable, offline-aware, install-only scope |

## Languages

English · हिन्दी · मराठी — fonts: Poppins (EN), Noto Sans Devanagari (HI/MR).
